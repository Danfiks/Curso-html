Meteodos de envio de formulario

Quando vc esta no padrao de envio de formualario eles aparecem na url

existem 2 metodos, Se nao se escreve nada se usa o meotodo "GET" , o outro metodo pode usar o metodo POST

quando se usa POST aparce em cima, o arquivo PHP mas mesmo assim , ele nao é seguro, se for em insepcionar e network vc descobre em name --> header --> cadastro.php o que foi enviado.

o unico jeito de proteger mesmo o dados precisa criptografar os dados e usar https

quando usar cada um?

quando vc quer compartilhar se usa o GET. Quando vc nao quer compartilhar dados sensiveis

quando vc quere um pouco mais de segurança se usa o metodo POST

quando se manda por GET pode enviar ate 3mil bytes , ou seja 3 mil letras, tambem nao pode se usar envio de dados e etc.

nao existe mais grande diferença de velociade entre GET e POST
